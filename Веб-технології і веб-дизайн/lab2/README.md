## Завдання 1

**Умова:**  
Напишіть PHP-скрипт, який обчислює площу сектора круга за формулою:
S = (α * r²) / 2

де α = 30 (градусів), r = 7.

```php
<?php
$alpha = 30; 
$r = 7;      

$sectorArea = ($alpha / 360) * pi() * pow($r, 2);

echo "Площа сектора круга: " . $sectorArea;
?>
```
